# MAX-DETOX-LIFESTYLE-BLACKOUT v1.0

Pi-hole gravity blocklist designed to support healthier lifestyle choices by blocking access to domains associated with unhealthy habits and financial blackholes.

## What's Blocked

This blocklist targets:

### Food Delivery & Aggregators
- DoorDash (doordash.com and subdomains)
- UberEats (ubereats.com and subdomains)
- GrubHub (grubhub.com and subdomains)

### Ultra-Processed Food Giants
- McDonald's (mcdonalds.com and subdomains)

## How to Use

1. Log in to your Pi-hole admin interface
2. Navigate to **Group Management** → **Adlists**
3. Add the following URL:
   ```
   https://raw.githubusercontent.com/Timkey/pihole-max-detox-lifestyle-blackout/main/blocklist.txt
   ```
4. Click **Add**
5. Update gravity by going to **Tools** → **Update Gravity**

## Purpose

This blocklist is intended to help individuals:
- Break unhealthy food ordering habits
- Reduce ultra-processed food consumption
- Avoid impulsive food delivery purchases
- Support lifestyle detox and health improvement goals
